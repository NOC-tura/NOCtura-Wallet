# Browser Extension

Noctura Wallet as a browser extension for Chrome/Firefox/Edge.

## Features

- Wallet creation and import
- Send/receive transactions
- Token management
- Swap interface
- Staking
- Compliance tools
- DApp integration

## Building

```bash
pnpm build
```

## Installation

### Chrome
1. Open `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select `dist` folder

## Development

```bash
pnpm dev
```

Extension reloads on changes.
