# API Server

Main REST API server for Noctura Wallet.

## Features

- Wallet operations
- Transaction management
- Token information
- Swap quotes
- Compliance endpoints
- Staking interface
- Governance API

## Setup

```bash
pnpm install
cp .env.example .env
pnpm dev
```

## API Documentation

API documentation available at `http://localhost:3001/api-docs`

## Environment Variables

See `.env.example` for required variables.
