# Compliance Package

Regulatory compliance and auditing tools.

## Features

- **View Keys**: Allow selective transaction inspection
- **Audit Tokens**: Generate compliance proofs
- **KYC Integration**: Know-Your-Customer verification
- **Travel Rule**: VASP protocol support
- **Geo-Fencing**: Jurisdiction restrictions
- **Compliance Reporting**: Regulatory reports and logs

## Compliance Tools

- **Selective Disclosure**: Share specific transaction details
- **Threshold Monitoring**: Alert on transaction thresholds
- **Risk Scoring**: Assess transaction risk
- **Audit Trail**: Complete transaction history

## Structure

```
src/
├── audit/         # Audit and view keys
├── kyc/          # KYC verification
├── threshold/    # Threshold monitoring
├── travel-rule/  # VASP protocol
├── geo-fencing/  # Jurisdiction control
└── reporting/    # Compliance reports
```
